# Aplaudia

Web publica de Aplaudia: estudio digital para creacion y mejora de paginas web, presencia digital y contenido con IA para productos, webs y presentaciones.

## Reglas de control operativo

Este repo contiene tambien la memoria tecnica ligera del proyecto, siguiendo el sistema usado en Aventuras Pixeladas, Cronoras y Arik Custom.

- Mantener los documentos `.md` compactos y utiles.
- No guardar secretos, tokens, contrasenas, datos personales sensibles ni datos fiscales.
- No guardar datos reales de clientes salvo que Carlos lo pida expresamente.
- `LAST_CODEX_REPORT.md` se reemplaza en cada tarea importante.
- El historico detallado queda conservado en Git.
- Codex debe leer estos documentos antes de tocar el proyecto.
- ChatGPT puede leer estos documentos para preparar prompts cerrados y continuar el trabajo sin perder contexto.

## Documentos clave

- `PROJECT_STATE.md`: estado confirmado del proyecto.
- `DECISIONS.md`: decisiones vigentes y limites.
- `WORKFLOW.md`: forma de trabajo ChatGPT / Codex / GitHub / Railway.
- `NEXT_TASK.md`: siguiente tarea recomendada.
- `LAST_CODEX_REPORT.md`: ultimo resumen de ejecucion; se sobrescribe en cada tarea.
